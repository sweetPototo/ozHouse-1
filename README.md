# 📗 Project 1 . Oz의 집(1차)

- 프로젝트 기간 : 2023. 12. 17 ~ 2024. 01. 19 (34일)
- 역할 : 팀장
- 참여 인원 : 4명

## 1. 프로젝트 소개

### 개발 환경

Java8, STS3, Maven, Oracle, DBeaver

### 기술 스택

**[ Back-end ]**

Mybatis, Spring Security, Qauth2.0, JavaMail API, 사업자등록번호상태조회 API

**[ Front-end ]**

JavaScript, JSP, HTML, CSS, Ajax
